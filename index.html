HTML

  • This is an example of an unordered list using the ul and li tags.
  • The declaration tells the browser what type of document type is should expect.
  • The head element contains information, or metadata, about the webpage.
  • The body element represents the visible content shown to the user.
  • An element refers to the entire element from opening tag to closing tag, whereas a tag refers only to what appears inside the angle brackets.
  • Elements provide the structure of the document, attributes help define those elements.

CSS

  • CSS defines the style of the webpage.
  • There are three ways to style a webpage using CSS: inline, internal, and external.
  • All have their place, but external is the cleanest method and allows the easiest design changes.
  • CSS rule consist of a selector, a property, and values.
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.

Git

  • Git is a version control system that lets you manage and keep track of each version of your project
  • Git commands always start with git and are used to save and track versions of your code
  • git status: checks what branch we are currently working in.
  • git checkout -b 'branch name': creates a new branch and switches to it.
  • git pull origin main: pulls source code and checks for commits to make sure we are working on the most current code.

JaveScript

  • JavaScript is a programming language that allows users to interact with websites
  • JavaScript is case sensitive. This means myVariable is not the same as myvariable
  • A variable is a named container that allows us to store data in our code.
  • Different type of variables include strings, numbers, boolean, arrays, and objects.
  • Comments can be written using // if no line break exists. Or /* */ if a line break exists.
  • Functions are resusable blocks of code that contain instructions to preform a specific task.
  • Control flow is the order in which a computer executes code in a script.
